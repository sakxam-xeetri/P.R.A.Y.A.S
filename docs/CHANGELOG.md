# Changelog

All notable changes to the PRAYAS V1 robotics platform documentation and codebase will be documented in this file.

## [1.0.0] - 2026-06-27

### Added
- Complete MkDocs documentation configuration using Material theme.
- System Architecture definitions including Distributed ESP32 Node specs.
- Electrical Wiring Diagrams and Power Distribution calculations.
- Mechanical dimensions, joint configuration, and frame layout.
- AI Node specifications (Xiaozhi voice assistant and Model Context Protocol).
- Base drivetrain specification utilizing 4x Johnson 12V 200RPM motors and dual BTS7960 drivers.
- Inter-Node Communication layout using low-latency ESP-NOW.
- Comprehensive BOM (Bill of Materials) and Manufacturing Assembly Guide.

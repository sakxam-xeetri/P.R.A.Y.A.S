# Network Signal Strength & Performance Test

## Purpose
This document details the test plan to verify Wi-Fi and Bluetooth signal strength, range, and latency.

## Test Procedure
1.  **RSSI Mapping**: Move the robot away from the Wi-Fi AP. Log the Received Signal Strength Indicator (RSSI) value at 5-meter increments up to 30 meters.
2.  **Packet Delivery Rate**: Verify that the packet drop rate remains below 1% under normal operating conditions.

# Manual Controls Panel

## Purpose
This document details the manual controls panel, which allows users to drive the base and trigger gestures.

## Control Interfaces
*   **Virtual Joysticks**: NippleJS virtual joysticks allow driving the motorized base and panning the camera.
*   **Quick Gestures**: Buttons to trigger preconfigured servo animations (e.g. wave, point, look-around).
*   **Emergency Stop**: A prominent red button that immediately halts all motors and joints.

# System Monitoring & Diagnostics

## Purpose
This document details the diagnostics screen, which shows the status and connection health of all nodes.

## Diagnostic Checks
The diagnostics panel queries the status of the nodes:
*   **Node Handshake Status**: Green/Red indicator showing node connection state.
*   **Memory Health**: Displays free heap and stack allocations.
*   **Packet Count**: Logs the number of sent/received ESP-NOW packets and tracks transmission errors.

# Telemetry & System Status Panel

## Purpose
This document details the telemetry panel, explaining how sensor and battery data are visualized.

## Display Elements
The telemetry panel updates at 5 Hz and displays:
*   **Battery Gauge**: High-contrast progress bar showing voltage, current draw, and charge percentage ($0	ext{ to }100\%$).
*   **System Temperature**: Visual thermometer displaying internal electronics bay temperature.
*   **Sonar Distance Maps**: High-contrast radar charts showing distances to obstacles.

# 3D Printing Parameters & Slicing Profile

## Purpose
This document details the slicing settings and printing parameters for the robot's 3D-printed structural components.

## Printing Configurations Table
| Parameter | Structural Parts (PETG) | Cosmetic Covers (PLA+) |
| :--- | :--- | :--- |
| **Nozzle Temperature** | $240^\circ	ext{C}$ to $245^\circ	ext{C}$ | $210^\circ	ext{C}$ to $215^\circ	ext{C}$ |
| **Bed Temperature** | $80^\circ	ext{C}$ | $60^\circ	ext{C}$ |
| **Infill Density** | 40% (Gyroid) | 15% (Grid) |
| **Perimeters (Walls)** | 4 | 2 |
| **Layer Height** | 0.20 mm | 0.20 mm |
| **Cooling Fan** | 30% (for strength) | 100% (for detail) |

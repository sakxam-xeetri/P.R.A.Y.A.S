# Robot Physical Dimensions

## Purpose
This document details the physical dimensions and clearances of the PRAYAS V1 robot.

## Physical Specifications

```
                     +--- [ Head Width: 160 mm ] ---+
                     |                              |
                     +----[ Torso Width: 320 mm ]---+
                                    │
                                    │
                                    │ (Torso Height: 700 mm)
                                    │
                                    │
                     +----[ Base Width: 350 mm ]----+
                     |                              |
                     +--- [ Base Length: 400 mm ]---+
```

*   **Total Height**: $1150	ext{ mm}$ (approx. $1.15$ meters).
*   **Base Plate footprint**: $400	ext{ mm} 	imes 350	ext{ mm}$ rectangular deck.
*   **Base Height (including wheels)**: $160	ext{ mm}$ off the ground.
*   **Spine Column (PVC)**: $700	ext{ mm}$ height, $4.5	ext{ in}$ outer diameter ($114.3	ext{ mm}$).
*   **Shoulder-to-Shoulder Span**: $320	ext{ mm}$.
*   **Arm Length**: $470	ext{ mm}$ (Shoulder-to-Elbow: $250	ext{ mm}$, Elbow-to-Wrist: $220	ext{ mm}$).
*   **Ground Clearance**: $30	ext{ mm}$ under the lowest base bracket.

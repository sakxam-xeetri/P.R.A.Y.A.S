# Robot Frame Design & Construction

## Purpose
This document describes the design and construction of the robot's physical frame, detailing the assembly of the base, column, and upper body.

## Frame Design Layout

```
                  [ Head Shell ]
                        │
                [ Neck Mount Plate ]
                        │
             [ 3D Printed Shoulder Frame ]
              /                                [ Left Arm ]                 [ Right Arm ]
              \                         /
            [ Torso PVC Mounting Adapter ]
                        │
             [ Torso Column (PVC Pipe) ]
                        │
             [ Base Flange Attachment ]
                        │
             [ Plywood Chassis Base ]
             /        |       |                [Motor 1] [Motor 2] [Motor 3] [Motor 4]
```

## Construction Details
*   **Base Plate Dimensions**: 400 mm × 350 mm rectangular deck cut from 12 mm birch plywood.
*   **PVC Mount**: The 70 cm PVC pipe is secured to the base using a 4-inch PVC floor flange, bolted to the plywood deck with M5 machine screws and wide washers to distribute load.
*   **Shoulder Mount**: A custom 3D-printed adapter slides into the top of the PVC pipe and is secured with through-bolts, supporting the upper torso panels.
